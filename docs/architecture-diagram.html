System Architecture

Mandera Analytics Pipeline

Data generation runs independently via GitHub Actions on a cron schedule, pushing synthetic records into MongoDB Atlas. The analytics pipeline is orchestrated by Apache Airflow — extracting, validating, transforming, and loading into the warehouse.
